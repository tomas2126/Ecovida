##  Descripción del proyecto
Ecovida es una aplicación web básica que simula una plataforma de apoyo a la conservación ambiental.  
Incluye:

- Página de inicio  
- Página de registro  
- Página de login  
- Página de donaciones  
- Archivo JavaScript con funciones simples  
- Archivo CSS para el estilo

##  Cómo ejecutar el proyecto

1. Descargar o clonar el repositorio:

2. Abrir la carpeta en Visual Studio Code.

3. Instalar la extensión **Live Server** (si no está instalada).

4. Abrir el archivo `index.html`.

5. Ejecutar:
- Clic derecho → **Open with Live Server**, o  
- Botón **Go Live** en VS Code.

El sitio se abrirá en el navegador.
